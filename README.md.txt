# Sistema Web - Gestão de Produtos

Sistema desenvolvido em PHP com PDO e MySQL.

## 🚀 Funcionalidades

- Login (simples)
- Cadastro de produtos
- Cadastro de fornecedores
- Listagem com AJAX
- Cesta de produtos
- Carrinho com total
- Limpar carrinho

## 🛠 Tecnologias

- PHP (PDO)
- MySQL
- HTML, CSS, JavaScript
- Bootstrap

## ⚙️ Como rodar

1. Inicie o XAMPP
2. Acesse:
http://localhost/apointernet/instalar.php
3. Depois:
http://localhost/apointernet
