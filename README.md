# 🛒 Sistema Web de Gestão de Produtos

Sistema web completo desenvolvido em PHP utilizando PDO e MySQL, com interface moderna em Bootstrap e funcionalidades dinâmicas com AJAX.

## 🚀 Demonstração

Sistema permite:

- Cadastro de fornecedores
- Cadastro de produtos vinculados
- Listagem dinâmica (AJAX)
- Seleção de produtos (cesta)
- Carrinho com cálculo de total
- Limpeza do carrinho

## 🛠 Tecnologias utilizadas

- PHP (PDO)
- MySQL
- HTML5 / CSS3
- JavaScript (Fetch API)
- Bootstrap 5
